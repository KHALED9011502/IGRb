# Discussion Posting Setup Guide

## How to Enable Posting

Follow these steps to create and post in the Discussion area:

### Step 1: Sign Up / Sign In
1. Click **"Sign Up"** or **"Sign In"** in the top right
2. Create an account or log in with existing credentials
3. Choose your preferred language (Arabic/English)

### Step 2: Enter Access Code
1. Look for the **ðŸ”’ Code** button in the top-left navbar (next to IGRb logo)
2. Click it to open the code modal
3. Enter the posting code: **9011502**
4. Click "Verify"
5. You should see success message: "Posting enabled"

### Step 3: Navigate to Discussion
1. Click the **"Discussion"** tab in the navbar
2. You should now see a **"Create Post"** button in the top right
3. Click it to create a new post

### Step 4: Create a Post
1. Fill in:
   - **Post Title**: Main idea/question
   - **Content**: Details about the post
   - **Platforms**: Select which platforms (PC, PS5, iOS, etc.)
2. Click **"Save"** to publish

### Step 5: Manage Posts
- **Edit**: Click the edit icon (pencil) on your own posts
- **Delete**: Click the delete icon (trash) to remove a post
  - Authors can delete their own posts
  - Admins can delete any post

## Post Display Format

Each post shows:
- **Title** (top-left, main idea)
- **Content** (post details below title)
- **Date** (when post was created)
- **Platforms** (displayed as colored badges)
- **Comments Stats** (bottom-right):
  - Comments today
  - Total comments

## Troubleshooting

### "Create Post" button doesn't appear
- Make sure you've entered code **9011502**
- Check browser console (F12) for error messages
- Try refreshing the page after entering code

### Can't enter code
- Make sure you're **signed in first**
- Try clearing browser cache and signing in again

### Post doesn't save
- Check browser console for error messages
- Make sure all fields are filled
- Verify internet connection

### Looking for existing posts?
- Posts show up immediately after creation
- Newest posts appear at the top
- Check if you're viewing the Discussion tab
